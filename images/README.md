## Cómo descargar desde Instagram

1. Abre Instagram en el navegador: https://www.instagram.com/familyfarm_bolivia/
2. Abre la foto que quieres descargar
3. Clic derecho → "Guardar imagen como..."  
   (En móvil: mantén presionada la foto → Guardar)
4. Renómbrala con el nombre exacto de la tabla de arriba
5. Colócala en esta carpeta `images/`

## Notas

- Todos los formatos funcionan: `.jpg`, `.jpeg`, `.png`, `.webp`
- Si una imagen falta, el sitio muestra un fondo de color de marca — no se rompe
- Para mejores resultados, usa fotos de buena iluminación y fondo limpio
- Las fotos cuadradas funcionan mejor para el grid de Instagram y los productos
